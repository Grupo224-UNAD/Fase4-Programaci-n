# Sistema Integral de Gestión de Clientes, servicios y reservas
Proyecto desarrollado en Python para la empresa ficticia Software FJ, aplicando Programación Orientada a Objetos (POO) y manejo avanzado de excepciones.

El sistema permite gestionar clientes, servicios y reservas sin utilizar bases de datos, implementando almacenamiento mediante objetos, listas internas y archivos de registro de eventos.

---

# Objetivo del proyecto

Desarrollar una aplicación modular, extensible y estable que permita:

- Gestionar clientes.
- Gestionar reservas.
- Gestionar distintos tipos de servicios.
- Validar información ingresada.
- Manejar errores sin detener la ejecución del sistema.
- Registrar eventos y errores mediante archivos logs.

---

# Características principales

- Registro de clientes.
- Creación y procesamiento de reservas.
- Manejo de distintos servicios:
  - Reserva de salas.
  - Alquiler de equipos.
  - Asesorías especializadas.
- Validaciones estrictas.
- Manejo avanzado de excepciones.
- Registro automático de eventos y errores.
- Arquitectura orientada a objetos.
